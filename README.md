# instagram-saved-to-excel
인스타그램 저장목록 HTML → 엑셀 변환기

인스타그램을 사용중이신가요?

인스타그램에서 관심 있는 게시물&릴스를 몇백~몇천개씩 저장하다보면
막상 예전에 저장한 게시물&릴스를 찾아보려 할 때
1. 저장목록이 너무 많아서 원하는 게시물&릴스가 어떤 것이었는지 헷갈리고
2. 느린 로딩시간을 견디며 한참동안 스크롤을 내렸던 경험
다들 한 번쯤은 있으셨을 텐데요,

그런 분들을 위해 내 인스타그램 '저장됨' 목록을 엑셀로 보기 편하게 내보내는 프로그램을 배포합니다.


## 📥 Step 1 — 인스타그램 데이터 내보내기

1. 인스타그램 앱에서 오른쪽 하단 **프로필 사진** 탭
2. 오른쪽 상단 **햄버거 메뉴(☰)** 탭
3. **계정 센터** → **내 정보 및 권한** 탭
4. **내 정보 내보내기** → **내보내기 만들기** 탭
5. 내보낼 프로필 선택 → **다음**
6. **외부 서비스로 내보내기** 선택
7. 내보낼 외부 서비스 선택 후 연결 → **다음**
8. 내보낼 빈도 선택 → **다음**
9. 비밀번호 입력 → **다음**
10. 이메일 알림 형식 등 설정 후 **내보내기 시작**

> ⏳ 내보내기 링크를 이메일로 받는 데 최대 30일이 소요될 수 있어요.
> 준비되면 이메일 + 인스타그램 알림으로 안내가 와요.
> 다운로드 가능한 링크는 **4일 이내**에 받아야 해요!

---

## 📂 Step 2 — HTML 파일 준비

전송받은 파일 중에서 **컬렉션** 정보가 담긴 HTML 파일을 찾아요.

> 💡 컬렉션 = 인스타그램 '저장됨'에서 직접 만든 카테고리 폴더

---

## ▶️ Step 3 — 변환 실행

찾은 HTML 파일을 `인스타그램_저장목록_변환.bat` 위에 **드래그 앤 드롭**하면
엑셀 파일이 자동으로 생성돼요! 🎉
